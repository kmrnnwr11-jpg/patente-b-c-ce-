# Icone teoria

Inserisci in questa cartella le icone personalizzate per gli argomenti della teoria.

Salva i file con questi nomi (formato PNG consigliato):

- `pericolo.png` → Segnali di pericolo
- `divieto.png` → Segnali di divieto e fermata/sosta
- `obbligo.png` → Segnali di obbligo
- `precedenza.png` → Segnali di precedenza / precedenza incroci
- `indicazione.png` → Segnali di indicazione e pannelli integrativi
- `cantiere.png` → Segnali complementari / lavori in corso
- `segnaletica.png` → Segnaletica orizzontale
- `direzioni.png` → Norme di circolazione / autostrade
- `semaforo.png` → Semafori e segnali degli agenti
- `velocita.png` → Limiti di velocità e distanza di sicurezza

Puoi aggiungere ulteriori icone creando nuovi file PNG e aggiornando la mappa in `src/pages/TheoryPage.tsx`.
